# DevFlow

A developer productivity platform for managing projects, tasks, and code snippets.

## Overview
DevFlow helps developers organize their work by providing:
- Project & task management
- Code snippet library with syntax highlighting
- Productivity analytics
- GitHub integration

## Tech Stack
- **Backend:** ASP.NET Core 8.0 Web API
- **Frontend:** Angular 17+
- **Database:** SQL Server
- **Authentication:** JWT
